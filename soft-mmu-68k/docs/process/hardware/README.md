# Hardware Process Notes

This directory contains process documentation for repeatable hardware-facing
work on the SM68861 / soft-MMU project.

Hardware process notes define how evidence should be prepared, captured, and
reported. They do not by themselves claim new FPGA, board, timing, or
compatibility results.
